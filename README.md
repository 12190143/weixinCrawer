# Cralwing for WeChat of public acount

![Travis](https://img.shields.io/badge/build-passing-red.svg)
![Language](https://img.shields.io/badge/Language-JAVA-green.svg)
![license](https://img.shields.io/dub/l/vibe-d.svg)

This is a tool for crawling weixin of public acount.This is a single-threaded programs.If you want it to be more effective,you can develop it base on this crawler. 

![weixin](http://ww3.sinaimg.cn/bmiddle/7bc698cdgw1eym9ezo1ixj206y05uq2v.jpg)

## QuickStart 
As follow, it is the program tree directory.

```java
.
├── cn
│   └── ecnu
│       └── weixin
│           ├── crawler
│           │   └── weixinCrawler.class
│           ├── search
│           │   └── SearchOnWeb.class
│           └── util
│               ├── Configuration.class
│               ├── CreateFloderFile.class
│               ├── TheCrawlerUtil.class
│               └── WebConstructor.class
└── log4j.properties
```

- weixinCralwer.class
	- this class file is launch class.

- SearchOnweb.class
	- This class is core of weixinCrawler.
	- [Sogou](http://weixin.sogou.com/) provide WeChat public acount search function.weixinCrawler get public acount data by using sogou search of WeChar channel.

- WebConstructor.class
	- webConstructor configure the browser.

### Runtime environment
- JDK 1.7
- Linux or Window OS

### Install

Install weixinCrawler

```shell
git clone https://github.com/huangbaoquan/weixinCrawler.git
```
### Run
you can package this program into jar. I have packaged this application and named it *weixinCralwr.jar*.You can get it in the project folder.And run it as follow:

```java
java -jar weixinCrawler.jar
```
by the way,weixinCrawler collects ten articles which WeChat of public acounts published recently. weixinCrawler needs run once a day for crawling updated data.You'd better set the run time at midnight.
